# 갑우문화사 생산현황 조회 서비스

## 절대 금지 사항

### 원본 엑셀 파일 수정 금지 (최우선 규칙)

**`생산진행현황/` 폴더 내의 엑셀 파일을 절대로 수정, 삭제, 이동, 덮어쓰기 하지 마세요.**

- `생산진행현황/재단지시서_2026년.xlsx` — 절대 수정 금지
- 이 파일은 실제 업무에서 사용되는 생산관리 원본 데이터입니다
- 읽기(Read)만 허용됩니다
- 테스트용으로도 원본 파일을 수정하지 마세요
- 변환 스크립트(convert.py)는 읽기 전용으로만 접근합니다

**이 규칙을 위반하면 실제 업무에 심각한 문제가 발생합니다.**

## 회사 정보

- **갑우문화사**: 출판 + 패키지(화장품) 인쇄 전문기업
- **2개 사업부**: 출판사업부(영업1~3팀), 패키지사업부(화장품 단상자/쇼핑백)
- **설비**: Heidelberg/Roland/Komori 인쇄기 8대 (10색, 5색, 2/2색)
- **생산공정**: 프리프레스(CTP) → 인쇄 → 코팅/박/실크 → 톰슨(타발) → 접착/제본 → 검수/납품
- **출판 주요 거래처**: 이투스에듀, 동행복권, HSAD, 에듀윌, 필통북스
- **패키지 주요 거래처**: 코스맥스, 코스메카코리아, 정샘물뷰티, 에뛰드, 지담디앤피
- **관계사**: 더원프린팅(제본), 프린트뱅크(중철)
- **현황(2026)**: 소량다품종 추세, 로트크기 감소, 가동률 30~50%, ERP 자체개발 중

## 프로젝트 개요

- 갑우문화사의 생산현황을 모바일에서 조회하는 정적 웹서비스
- 엑셀 → JSON 변환 → 정적 HTML로 영업사원이 거래처 현황을 확인

## 구조

```
gabwoo/
├── 생산진행현황/           # 원본 데이터 (절대 수정 금지)
│   └── 재단지시서_2026년.xlsx
├── convert.py              # 엑셀 → JSON 변환기 (읽기 전용 접근)
├── web/                    # 배포용 정적 파일
│   ├── index.html          # 모바일 웹앱
│   └── data.json           # 변환된 데이터
└── docs/                   # 기획 문서
```

## 데이터 정확성

- 이 프로젝트의 데이터는 실제 고객 응대에 사용됩니다
- 변환 결과는 반드시 원본 엑셀과 대조 검증해야 합니다
- 데이터가 틀리면 프로젝트 전체 신뢰를 잃습니다
